\# Titanic EDA



Exploratory Data Analysis of the Titanic dataset from Kaggle.



\## Objective

The goal of this project is to understand which factors affected passenger survival.



\## Steps Performed

\- Loaded and explored the dataset

\- Handled missing values

\- Created new features (FamilySize, IsAlone, Title)

\- Performed exploratory data analysis

\- Created visualizations using Matplotlib and Seaborn



\## Key Insights

\- Females had a much higher survival rate than males

\- First class passengers survived more than third class

\- Passengers traveling alone had lower survival chances

\- Children had better survival probability
